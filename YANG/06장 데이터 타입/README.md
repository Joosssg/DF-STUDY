# 06장 데이터 타입
- 데이터 타입마다 확보해야하는 메모리 공간의 크기, 메모리에 저장되는 2진수도 다르기 때문에 자바스크립트에서는 타입을 구분해서 값을 취급해야한다.

|  구분   |  데이터 타입   | 설명                             |
|:-----:|:---------:|:-------------------------------|
| 원시 타입 |  number   | 숫자, 정수, 실수 구분 없이 하나의 숫자 타입만 존재 |
|       |  string   | 문자열                            |
|       |  boolean  | true/false                     |
|       | undefined | 변수 초기값                         |
|       |   null    | 값이 없음                          |
|       |  symbol   | ES7에 추가된 7번째 타입                |
| 객체 타입 |  object   | 객체, 함수, 배열 등 |
