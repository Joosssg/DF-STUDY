# **19.1  객체지향 프로그래밍**
객체의 집합으로 프로그램을 표현하려는 프로그래밍 패러다임을 **객체지향 프로그래밍**이라고 한다.
<br>
객체의 다양한 속성 중 프로그램에 필요한 속성을 간추려 내어 표현하는 것을 **추상화**라고 한다.
