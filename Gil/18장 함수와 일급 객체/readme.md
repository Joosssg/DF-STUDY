# **18.1 일급 객체**

일급객체의 조건

* 무명의 리터럴로 생성할 수 있다.(런타임에 생성가능)
* 변수나 자료구조(객체, 배열)에 저장 가능하다.
* 함수의 매개변수에 전달 가능하다.
* 함수의 반환값으로 사용 가능하다.

> 함수는 4가지에 모두 해당한다.
